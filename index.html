Projets

Gestion de Temps et Paie

Un projet personnel visant à faciliter la gestion des temps de travail et des paies pour les ouvriers intérimaires.

Le backend est développé en PHP Laravel, et les données sont gérées via MySQL. L’application est déployée sur AWS et utilise Docker pour la conteneurisation.

Compétences : Développement d’Applications, Gestion des Bases de Données, Conduite de Projet.

Livraison entre Particuliers

Une application universitaire qui permet aux particuliers de gérer l’envoi et la réception de colis de manière sécurisée.

Le frontend est développé avec Vue.js, tandis que le backend repose sur PHP Laravel. Une base de données MySQL assure le stockage des informations.

Compétences : Développement d’Applications, Gestion des Bases de Données, Travail en Équipe.
